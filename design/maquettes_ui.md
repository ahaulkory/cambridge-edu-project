# Maquette de l'Interface Utilisateur - Site Web Éducatif Cambridge

## Thème général
- **Palette de couleurs** : Bleu Cambridge (#1E40AF) comme couleur principale, avec des accents en orange (#F97316) et vert (#10B981)
- **Typographie** : Police sans-serif moderne et lisible (Montserrat pour les titres, Open Sans pour le texte)
- **Style** : Interface épurée avec des éléments ludiques adaptés aux jeunes de 16 ans
- **Composants** : Cartes interactives, boutons animés, icônes modernes

## Éléments d'interface spécifiques

### Navigation
- Menu principal fixe en haut
- Fil d'Ariane pour la navigation dans les cours
- Boutons "Précédent" et "Suivant" pour les modules

### Composants interactifs
- Cartes de cours avec effet de survol
- Barres de progression colorées
- Boutons d'action avec animations subtiles
- Quiz et exercices interactifs

### Éléments ludiques
- Badges de réussite
- Animations de célébration pour les accomplissements
- Système de points ou d'étoiles pour la progression

## Adaptations mobiles
- Menu hamburger pour la navigation
- Disposition en une seule colonne
- Taille de texte adaptée pour la lisibilité
- Boutons plus grands pour faciliter le toucher
- Contenu priorisé pour l'expérience mobile
